# CozySwing Design System

## Brand Colors

| Name      | Hex       | Role                   |
|-----------|-----------|------------------------|
| Lavender  | `#9ba0dd` | Primary / main color   |
| Burgundy  | `#71002e` | Accent                 |
| Off-Black | `#1a1a1a` | Text / dark backgrounds|
| Off-White | `#efebe6` | Backgrounds / light UI |

## Layout

- **Responsive**: mobile-first, single breakpoint at `768px`
- **Header**: logo (left) + horizontal nav (right); collapses to hamburger menu on mobile
- **Footer**: contains links to legal pages (Imprint, Privacy Policy)
